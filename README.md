# Premium Habit Tracker

A modern, product-level upgrade for the Habit Tracker application using **React**, **Vite**, and **Firebase**.

## Features List
- **Advanced State Management**: Pure React state logic mapped seamlessly to Firebase and Local Storage.
- **Glassmorphic Design System**: Uses bleeding-edge premium aesthetic parameters optimized for focus. High contrast variables, smooth animations, layout transitions.
- **Dynamic Themes**:
  - `Dark`: The classic dark mode with a subtle glow.
  - `Light`: The refined minimalist mode for daytime use.
  - `Matrix`: Cyberpunk digital rain mode with fully custom styling and rendering loop.
- **Responsive Tracking**: Scrollable grids that scale from mobile viewports to ultra-wides, equipped with fixed-column labels.
- **Comprehensive Sync**: Persistent Google Auth configuration that allows device sync and anonymous guest mode.
- **CSV Data Export**: Single-click data dumps for local backup tracking.
- **SEO Optimized**: Re-configured for maximum reach with meta descriptions, updated scales, and caching.

## How to use
```bash
# Starts the development server at http://localhost:5173
npm run dev
```

For production builds:
```bash
npm run build
```
